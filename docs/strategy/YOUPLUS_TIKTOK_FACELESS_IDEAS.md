# YOU+ FACELESS TIKTOK CONTENT IDEAS

> **Goal:** Daily TikTok content without showing face, maintaining vulnerability/story
>
> **Formats:** App demos, text-on-screen, memes, screen recordings, hand-only shots
>
> **Principle:** Same story, different wrapper

---

## CORE STRATEGY

### Why Faceless Works for You+

**Your Story = The Hero**
- Product is weird enough to be visual
- The call interface IS the story
- Text overlays maintain vulnerable tone
- Faceless = focus on concept, not personality

**Faceless Advantages:**
- Can create 3-5 per day (no filming yourself)
- Repurpose same footage multiple ways
- Lower barrier = more volume
- Story remains, format changes

## PRODUCTION WORKFLOW (BATCH CREATE 7 DAYS IN 1 HOUR)

### Setup (Once)
1. **Record reusable assets:**
   - Screen recording of app (all screens)
   - Hand holding phone (various angles)
   - Call interface (ringing, answered, ended)
   - Notification pop-ups

2. **Tools needed:**
   - Screen recording app (built-in iOS or OBS)
   - CapCut or TikTok editor (for text overlays)
   - Sound library (epidemic sound, TikTok audio)

### Daily Batch Process (15 min/day)

**Monday Batch (Record 3 Concepts):**
1. Concept 1: App demo (screen recording)
2. Concept 2: Text meme (no video needed)
3. Concept 3: Hand-only shot

**Tuesday: Edit Monday's Videos**
- Add text overlays
- Add sound
- Schedule posts

**Repeat:** Record 3, edit 3, post 3 (continuous cycle)

## VIRAL POTENTIAL ANALYSIS

### Why These Ideas Can Go Viral

**1. Pattern Interrupts**
- Unusual concept (voice clone, AI accountability)
- Unexpected consequences (phone locks, voice apologies)
- Familiar pain (everyone's quit apps)

**2. Emotional Triggers**
- Shame (can't keep promises)
- Curiosity (how does this work?)
- FOMO (73/100 spots taken)

**3. Shareability**
- "This is so me" moment (identification)
- Weird enough to share ("look at this")
- Practical enough to want ("I need this")

**4. Comment Bait**
- Controversial (is this healthy?)
- Relatable (everyone has excuse graveyard)
- Invitation ("if you can't keep promises...")
